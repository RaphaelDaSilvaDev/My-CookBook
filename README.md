## To run on your machine
### Backend
```shell
  # Change to Backend Folder
  cd backend
  # Install Dependencies:
  npm i
  
  # Start Server:
  npm start
```

### Frontend
```shell
  # Change to Frontend Folder
  cd frontend
  # Install Dependencies:
  npm i
  
  # Start Server:
  npm start
```
